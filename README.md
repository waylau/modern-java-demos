## 内容简介


现代Java案例大全，包含从 Java 8 到 Java 18 所有新特性及其案例。学习 Java 的同志们，燥起来！

技术版本如下：

* JDK 18
* Eclipse IDE for Java Developers 2020-03 (4.15.0)
* JUnit 5.5.1
* ...

## 源码

所有示例源码，移步至<https://github.com/waylau/modern-java-demos/>  `samples` 目录下。

工程代码可以直接导入Eclipse IDE。
